main() — Точка входа в программу, запуск основного окна и инициализация приложения.
run() — Запуск основного цикла работы окна или игрового процесса.
processEvents() — Обработка событий окна (например, нажатий клавиш, мыши и т.д.).
update() — Обновление состояния объектов на экране (игрок, враги, ловушки и т.д.).
render() — Отрисовка всех объектов на экране.
handleButtonClick(const sf::Vector2f& mousePos) — Обработка нажатия на кнопки интерфейса по координате мыши (например, «Войти», «Регистрация», «Выход» и др.).
collect() — Сбор ключа игроком.
setPosition(float x, float y) — Установка позиции объекта (игрока, врага, ловушки и т.д.) на экране.
updateAnimation() — Обновление анимации объекта (например, ключа или игрока).
handleTextInput(sf::Event& event) — Обработка ввода текста пользователем (например, ввод логина или пароля).
draw(sf::RenderWindow& window) — Отрисовка объекта (игрока, врага, ловушки и т.д.) в окне.
takeDamage(int damage) — Нанесение урона объекту (например, игроку или врагу).
heal(int amount) — Восстановление здоровья объекта (например, игрока).
setColor(const sf::Color& color) — Установка цвета объекта.
setSize(const sf::Vector2f& size) — Установка размера объекта.
setFillColor(const sf::Color& color) — Установка цвета заливки объекта.
move(float x, float y) — Перемещение объекта на заданное смещение по координатам.
move(const sf::Vector2f& offset) — Перемещение объекта на заданный вектор смещения.
checkWallCollisions() — Проверка столкновений объекта со стенами.
update(const sf::Vector2f& playerPosition) — Обновление состояния врага с учётом позиции игрока.
activate() — Активация ловушки.
deactivate() — Деактивация ловушки.
registerUser(const std::string& username, const std::string& password) — Регистрация пользователя с указанными логином и паролем.
loginUser(const std::string& username, const std::string& password) — Вход пользователя в систему по логину и паролю.
saveUserData(const std::string& username, const std::string& password) — Сохранение данных пользователя в файл.
loadUserData(const std::string& username, const std::string& password) — Загрузка данных пользователя из файла.
hashPassword(const std::string& password) — Хеширование пароля пользователя для безопасности.
addRecord(const std::string& username, int score, const std::string& difficulty, int mapNumber) — Добавление новой записи в таблицу рекордов.
loadFromFile() — Загрузка рекордов из файла.
clearRecords() — Очистка таблицы рекордов.
setState(WindowState state) — Установка текущего состояния окна (например, «Главное меню», «Игра», «Настройки» и др.).
initializeUIElements() — Инициализация элементов интерфейса (кнопки, поля ввода и др.).
handleEvents() — Обработка всех событий окна.
handleMainWindowEvents(const sf::Event& event) — Обработка событий для главного окна (например, нажатия на кнопки «Войти», «Регистрация» и др.).
handleLoginWindowEvents(const sf::Event& event) — Обработка событий окна авторизации (например, ввод логина и пароля, нажатие кнопки «Войти»).
handleRegisterWindowEvents(const sf::Event& event) — Обработка событий окна регистрации (например, ввод данных, нажатие кнопки «Зарегистрироваться»).
handleMainMenuEvents(const sf::Event& event) — Обработка событий главного меню (например, нажатия на кнопки «Начать игру», «Настройки», «Рекорды», «Выход» и др.).
handleMapSelectionEvents(const sf::Event& event) — Обработка событий окна выбора карты (например, выбор уровня сложности или карты).
handleCharacterSelectEvents(const sf::Event& event) — Обработка событий окна выбора персонажа (например, выбор «Медведь», «Динозавр», «Кот»).
handleSettingsEvents(const sf::Event& event) — Обработка событий окна настроек (например, изменение громкости, темы оформления и др.).
handleExitConfirmationEvents(const sf::Event& event) — Обработка событий окна подтверждения выхода (например, нажатие кнопок «Да» или «Нет»).
handleHighscoresEvents(const sf::Event& event) — Обработка событий окна рекордов (например, просмотр и возврат в меню).
handleSubmit() — Обработка подтверждения ввода данных (например, отправка формы регистрации или входа).
togglePasswordVisibility() — Переключение видимости пароля в поле ввода («Показать/скрыть пароль»).
enterMainMenu() — Переход в главное меню.
draw() — Отрисовка всех элементов текущего окна.
drawMainWindow() — Отрисовка элементов главного окна (например, кнопок «Войти», «Регистрация», «Выход»).
drawLoginWindow() — Отрисовка окна авторизации (поля ввода, кнопка «Войти» и др.).
drawRegisterWindow() — Отрисовка окна регистрации (поля ввода, кнопка «Зарегистрироваться» и др.).
drawMainMenu() — Отрисовка главного меню (кнопки «Начать игру», «Настройки», «Рекорды», «Выход» и др.).
drawMapSelection() — Отрисовка окна выбора карты (кнопки выбора сложности, карты и др.).
drawCharacterSelect() — Отрисовка окна выбора персонажа (кнопки выбора «Медведь», «Динозавр», «Кот»).
drawSettings() — Отрисовка окна настроек (ползунки громкости, кнопка «Сменить тему» и др.).
drawExitConfirmation() — Отрисовка окна подтверждения выхода (кнопки «Да», «Нет»).
drawHighscores() — Отрисовка окна рекордов (список рекордов, кнопка возврата).
playButtonClickSound() — Воспроизведение звука нажатия на кнопку.
playKeyCollectSound() — Воспроизведение звука сбора ключа.
run() — Запуск игрового процесса (основной цикл игры).
processEvents() — Обработка событий во время игры (например, нажатия клавиш управления).
update() — Обновление состояния игры (игрок, враги, ловушки и т.д.).
render() — Отрисовка всех игровых объектов.
loadTextureWithChromaKey(sf::Texture& texture, const std::string& filename, sf::Color chromaKey) — Загрузка текстуры с применением хромакея (удаление определённого цвета).
generateMazeDFS(std::vector<std::vector<int>>& maze, int width, int height, int x, int y, std::mt19937& rng) — Генерация лабиринта методом поиска в глубину.
loadMaze() — Загрузка лабиринта для текущего уровня.
handlePlayerInput() — Обработка ввода от игрока (например, управление персонажем).
checkCollision(const sf::FloatRect& rect) — Проверка столкновения объекта с другими объектами или стенами.
updateCamera() — Обновление положения камеры в зависимости от позиции игрока.
updateHealthBar() — Обновление состояния полосы здоровья.
placeTraps() — Размещение ловушек на уровне.
checkTrapCollisions() — Проверка столкновений игрока с ловушками.
updateDamageFlash() — Обновление эффекта получения урона (мигание экрана).
placeKeys() — Размещение ключей на уровне.
checkKeyCollection() — Проверка сбора ключа игроком.
initializePauseMenu() — Инициализация элементов меню паузы (кнопки «Продолжить», «Заново», «В меню»).
handlePauseEvents(const sf::Event& event) — Обработка событий меню паузы (нажатия на кнопки «Продолжить», «Заново», «В меню»).
drawPauseMenu() — Отрисовка меню паузы (кнопки «Продолжить», «Заново», «В меню»).
togglePause() — Включение/выключение режима паузы.
initializeGameOverWindow() — Инициализация окна окончания игры (кнопки «Заново», «В меню»).
drawGameOverWindow() — Отрисовка окна окончания игры (кнопки «Заново», «В меню»).
handleGameOverEvents(const sf::Event& event) — Обработка событий окна окончания игры (нажатия на кнопки «Заново», «В меню»).
